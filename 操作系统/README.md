# 操作系统面试题
## 1. 线程和进程的区别  

ME:  
---  
+ 进程是资源分配的最小单位；线程是程序运行的最小单位。  
+ 每个进程都有自己的堆空间；一个进程内的多个线程共享堆空间以及里面的资源。  
+ 由于进程都有自己的空间，所以进程切换需要消耗大量的资源；线程由于共享进程内存，所以切换资源消耗的资源较小。  
+ 进程间通信比线程通信复杂。  
+ 进程发生故障时，不会对其他进程产生影响；而线程发生故障时，会导致整个进程崩溃。
